Este projeto apresenta a aplicação de técnicas de Amostragem em Estatística, utilizando um conjunto de dados de imóveis para aluguel.
Foram realizadas:
Amostragem Aleatória Simples
Amostragem Estratificada
Discretização da área em categorias
Classificação do valor total por quartis (Q1, Q2, Q3, Q4)
One-Hot Encoding das categorias de valor
Para executar os códigos Python, as seguintes bibliotecas são necessárias:
Biblioteca 
pandas	Manipulação e análise de dados
numpy	Operações matemáticas e numéricas
random	Seleção aleatória de índices para amostragem
